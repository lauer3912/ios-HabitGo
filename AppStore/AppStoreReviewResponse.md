## 1. App Purpose Description

**Problem the app solves:**
HabitArcFlow helps users build lasting habits through visual streak tracking and reminder notifications.

**Value to users:**
- Simple, distraction-free habit tracking
- Visual streak motivation with flame indicators
- Monthly calendar view for historical tracking
- Fully offline — works without internet, no account required
- One-time purchase ($9.99), lifetime access

**Target audience:**
- Individuals looking to build better daily habits
- Users who prefer privacy-focused, offline apps

## 2. Feature Access Instructions

### First Launch
1. Launch app → See empty habit list with "Add your first habit" prompt
2. Tap the "+" floating action button (bottom right)

### Add New Habit
1. Enter habit name (e.g., "Morning Exercise", "Read 30 minutes")
2. Select icon from 16 SF Symbol options (checkmark, book, dumbbell, moon, heart, star, flame, leaf, brain, bed, coffee, pill, lightning, etc.)
3. Choose color from 8 accent colors
4. Select frequency: Daily, Weekdays, Weekends, or Weekly
5. Optionally enable daily reminder notification
6. Tap "Save"

### Complete a Habit
1. View today's habits in the main list
2. Tap the circular checkbox on the left of a habit
3. Checkbox fills with accent color → Streak increments
4. Flame icon appears showing current consecutive days

### View Statistics
1. Scroll to bottom of habit list
2. View "Current Streak" and "Longest Streak" for each habit
3. Total completions count shown

### Calendar View
1. Tap the calendar icon in the navigation bar
2. Monthly calendar shows colored dots for completed days
3. Tap any date to see which habits were completed

### Settings
1. Tap "Settings" tab
2. Options: Reset all data, About
