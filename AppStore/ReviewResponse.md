## 2. App Purpose Description

**Problem the app solves:**
Many people struggle to build lasting habits due to lack of accountability and difficulty tracking progress. HabitArcFlow provides a simple, beautiful way to track daily habits with streak-based motivation.

**Value to users:**
- Build lasting habits through visual streak tracking
- Stay motivated with flame indicators showing consecutive days completed
- See progress at a glance with color-coded habit icons
- View detailed statistics: current streak, longest streak, total completions
- Monthly calendar view shows historical habit completion at a glance
- Fully offline — works without internet, no account required

**Target audience:**
- Individuals looking to build better daily habits
- Users who prefer simple, distraction-free habit tracking
- People who want to track progress without sharing data online

## 3. Feature Access Instructions

**No login or account required. App is fully functional offline.**

### Core User Flows:

**Flow 1: Add a New Habit**
1. Launch app → See empty habit list (first launch)
2. Tap the "+" floating action button (bottom right)
3. Enter habit name (e.g., "Exercise", "Read 30 min")
4. Select icon from emoji picker
5. Choose frequency: Daily, Weekdays, Weekends, or Weekly
6. Tap "Save" → Habit appears in list

**Flow 2: Complete a Habit**
1. Launch app → See habit list with today's habits
2. Tap the circular checkbox on the left of a habit
3. Checkbox fills with accent color → Streak increments
4. Flame icon appears showing current streak count

**Flow 3: View Statistics**
1. Scroll to bottom of habit list
2. View "Current Streak" and "Longest Streak" for each habit
3. Tap a habit row to see detailed stats

**Flow 4: Calendar View**
1. Tap the calendar icon in the navigation bar
2. Monthly calendar shows colored dots for completed days
3. Tap any date to see which habits were completed

**Flow 5: Settings**
1. Tap "Settings" tab
2. Options include: Reset all data, Export data, About
